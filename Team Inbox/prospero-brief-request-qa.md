# Research Brief Request

**From:** Adama
**To:** Prospero

**Subject:** Candidate profile needed — QA & Data Quality Specialist

## The Need

As the job hunting pipeline grows (harvester → scorer → applications → gmail monitor), we need someone who validates that each stage is working correctly — data quality, pipeline integrity, selector health, and cross-project state consistency.

The immediate trigger: the harvester returned 7 jobs per collection when 25 were visible on page 1. A QA specialist would have caught this before it became a question from the owner.

## Scope of Role

- Validate harvester output (correct counts, field completeness, deduplication)
- Check pipeline handoffs (jobs → scores → applications → gmail events)
- Inspect data quality in SQLite — nulls, malformed fields, broken foreign keys
- Design and run test scenarios for new features before they go live
- Flag anomalies clearly so the right specialist can fix them

## What to Research

Profile a senior QA / data quality engineer. What does genuine expertise look like in testing data pipelines, web scrapers, and AI-assisted workflows?

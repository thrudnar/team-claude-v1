# Research Brief — Prompt Engineer

**From:** Adama
**To:** Prospero
**Date:** 2026-04-01

## Request

Research and define the ideal candidate profile for a Prompt Engineer joining the Team Claude AI team.

## Context

We are building a job description match scoring system (Project 2). The core of that system is a Claude prompt that evaluates each job listing against the owner's background and returns structured scores. The prompt needs to:

- Produce consistent, calibrated output across many job descriptions
- Improve over time through iteration — with each version stored in the DB alongside its scores
- Be benchmarked against a set of human-reviewed sample assessments (ground truth the owner will provide)
- Eventually generate cover letter drafts for high-scoring roles

This role owns the prompt. They are not a software engineer — Prospero and Argus handle code. They are a language and reasoning specialist who understands how to instruct a large language model precisely, evaluate output quality systematically, and iterate with discipline.

## Deliverable

A candidate brief covering: identity/name suggestion, role definition, skills & traits, tools/methods they'd use, and how they'd collaborate with the rest of the team.

Please deliver to `Team Inbox/` as `candidate-brief-prompt-engineer.md`.
